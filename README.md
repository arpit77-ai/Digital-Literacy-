# Digital-Literacy-
It contains information about digital literacy

This project was completed as part of the Digital Literacy course requirements for the Student
Digital Ambassador role. The primary objective was to build a comprehensive Digital Literacy
Portfolio that demonstrates proficiency in professional online presence, digital security, and
collaborative tools. This report documents the completion of five specialized tasks—ranging from
creating awareness infographics to researching cybercrime prevention—all organized within a
professional GitHub repository.


This project has significantly enhanced my digital literacy by forcing me to move beyond being a
passive user of technology to becoming an active, professional participant in the digital world. I
have learned how to manage my digital footprint, communicate professionally, and protect
myself against common online risks. These skills form the foundation of my professional identity
as a Computer Science student at VIT Bhopal.
